# Multi-Page website
This project demonstrates how to structure a multi-page website using only semantic HTML. It includes a homepage, projects page, articles page, and contact page, all connected through a shared navigation bar. 
The focus of this project is on HTML structure, semantic elements, and basic SEO meta tags, without any styling or JavaScript.
